# Lifewiki Daily Note Contract

- Vault root: `$LIFEWIKI_VAULT`
- Daily note path format: `-Daily-Notes/YYYY-MM-DD.md`
- Default section for authored note capture: `## Notes 📝`
- Explicit inbox section: `## Inbox 📥`
- Forbidden write targets: `## Logs 🪵` and query-driven dashboard blocks

Writers should append inside the target section without changing unrelated headings or task queries.
